./ai -m gpt-4 summarize "Your text"

-i, --interactive     Start interactive mode
--list               List all available tasks
-u URL               Use different API URL
-m MODEL             Use different model (gpt-3.5-turbo, gpt-4)
-t NUM               Max tokens (default: 1000)
--temperature NUM    Creativity 0-1 (default: 0.7)
-v, --verbose        Show detailed output
-h, --help          Show full help
